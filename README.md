# 💼 AI General Interview Coach

An interactive, AI-powered interview preparation chatbot built with **Python**, **Streamlit**, and the **Google GenAI SDK** (`gemini-2.5-flash`). This app serves as a virtual interviewer, asking classic behavioral and situational questions, evaluating your responses using corporate coaching best practices, and providing instant constructive feedback.

## ✨ Features
*   **Dynamic Interviewing:** Simulates a realistic interview conversation by asking relevant, general-purpose interview questions one at a time.
*   **Instant Feedback:** Evaluates user answers right away, highlighting strengths and offering 1–2 practical tips for improvement (leveraging the STAR method).
*   **Memory Context:** Tracks the conversation history to ensure smooth transitions between questions.
*   **Zero Local Setup:** Built to deploy straight from GitHub to Streamlit Community Cloud.

---

## 🛠️ Tech Stack & Architecture
*   **Frontend UI:** [Streamlit](https://streamlit.io/) (for a clean, responsive chat web interface)
*   **LLM Orchestration:** `google-genai` Python SDK
*   **AI Model:** `gemini-2.5-flash` (optimized for fast response times and natural dialogue processing)
